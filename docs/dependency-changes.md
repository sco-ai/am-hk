# 依赖变更审计日志

## 格式规范

```
## YYYY-MM-DD - 变更类型

**操作人**: @username
**原因**: 简述变更原因
**影响范围**: 受影响的服务/模块

### 变更内容
- 包名: 旧版本 -> 新版本

### 验证结果
- [ ] 单元测试通过
- [ ] 集成测试通过
- [ ] 生产环境验证
```
