# AM-HK 工作日志与记忆系统

**生效日期**: 2026-04-08  
**记录频率**: 每日至少 2 次  
**存储位置**: 
- 每日日志: `memory/YYYY-MM-DD.md`
- 长期记忆: `MEMORY.md`
- 工作记录: `logs/worklog-YYYY-MM-DD.log`

---

## 记录规范

### 每日两次总结时间点

| 时间 | 类型 | 内容 |
|------|------|------|
| 12:00 | 午间总结 | 上午完成事项、遇到的问题、下午计划 |
| 22:00 | 日终总结 | 全天完成事项、关键决策、明日计划 |

### 记录内容模板

```markdown
## 时间: HH:MM

### 完成事项
- [x] 事项1 (耗时: X分钟)
- [x] 事项2 (耗时: X分钟)

### 遇到的问题
- 问题描述: ...
- 解决方案: ...

### 关键决策
- 决策: ...
- 原因: ...

### 下一步计划
- [ ] 待办1
- [ ] 待办2
```

---

## 记忆分级

| 级别 | 存储位置 | 保留时间 | 示例 |
|------|----------|----------|------|
| L1 - 临时 | 每日日志 | 30天 | 调试信息、临时配置 |
| L2 - 短期 | 本周日志 | 90天 | 项目进展、会议纪要 |
| L3 - 长期 | MEMORY.md | 永久 | 架构决策、核心规则 |
