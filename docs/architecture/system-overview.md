# AM-HK v3.0 系统架构总览

## 核心原则

- ✅ 全部 Agent 不使用本地模型（RTX4090仅用于计算/缓存/数据）
- ✅ 全部采用 线上大模型 / API（可商用、可扩展）
- ✅ 多模型融合架构

## 数据层（多市场）

```
数据层（多市场）
├─ BTC（Binance / OKX）
├─ 港股（老虎证券 / 富途）
├─ 美股（老虎证券）
├─ 新闻（NewsAPI / Twitter / Reddit）

↓ Kafka（统一流）

Agent1 → Agent2 → Agent3 → Agent4 → Agent5 → 执行
   ↑                                            ↓
   └──────────── Agent6（学习反馈闭环）────────────┘
```

## 四大AI能力融合

| 能力 | 模型 | 作用 |
|------|------|------|
| 时间序列 | Informer / N-HiTS | 价格预测、趋势分解 |
| 强化学习 | PPO / DQN | 自动交易决策、仓位优化 |
| GNN | Temporal GNN | 市场联动、传染分析 |
| NLP | FinBERT / GPT-4.1 | 新闻情绪、宏观理解 |
