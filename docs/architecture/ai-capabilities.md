# 四大AI能力融合

## 1. 时间序列预测（核心）

| 模型 | 作用 |
|------|------|
| Informer | 长序列预测 |
| Autoformer | 趋势识别 |
| N-HiTS | 波动分解 |

**输出**:
- BTC未来走势
- 港股开盘方向预测

---

## 2. 强化学习交易代理

| 模型 | 作用 |
|------|------|
| PPO + LSTM | 自动交易决策 |
| DQN + CNN | K线形态识别 |
| A3C | 快速策略优化 |

**输出**:
- 仓位比例
- 止盈止损优化

---

## 3. 图神经网络（GNN）

| 模型 | 作用 |
|------|------|
| Temporal GNN | 市场传染 |
| GAT/GCN | 板块联动 |

**应用**:
- BTC → 美股 → 港股联动
- AI板块、科技股联动

---

## 4. NLP 情绪系统

| 模型 | 作用 |
|------|------|
| FinBERT | 新闻情绪 |
| GPT-4.1 | 宏观理解 |
| 社交媒体分析 | 市场热度 |

**数据源**:
- NewsAPI
- Twitter / Reddit
- 财报/公告
