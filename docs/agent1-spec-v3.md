- 多市场数据接入（BTC / 港股 / 美股 / 新闻）

# 跨市场信号引擎（Cross-Market Signal Engine）

## 🎯 核心逻辑

```
领先市场（Crypto）
        ↓
确认市场（美股）
        ↓
执行市场（港股）
```

👉 输出：港股可交易标的 + 方向 + 强度

---

## 三层信号体系（结构化）

### 🧠 Layer1：Crypto 先行信号（Lead Signal）

**核心标的**
- Bitcoin（风险总开关）
- Ethereum（科技/生态）
- Dogecoin（情绪）

**输出指标**
```
crypto_trend_score（趋势）
crypto_momentum（动量）
crypto_sentiment（情绪）
```

**判定逻辑（示例）**
```
BTC 5分钟动量 > +1% → 强风险偏好
ETH > BTC → 科技偏好增强
DOGE暴涨 → 散户情绪极强
```

### 🧠 Layer2：美股确认信号（Confirmation）

**核心映射标的**

加密链：
- Coinbase Global Inc.
- Marathon Digital Holdings

科技核心：
- NVIDIA Corporation
- Tesla Inc.

指数：
- NASDAQ Composite
- Invesco QQQ Trust

**输出指标**
```
us_trend_score
sector_strength（板块强度）
risk_on_off（风险开关）
```

**判定逻辑**
```
BTC ↑ + COIN ↑ + QQQ ↑ → 强确认（可交易）
BTC ↑ + 美股不动 → 信号减弱
```

### 🧠 Layer3：港股映射与执行信号（Execution Layer）

**映射规则（核心）**

#### 1️⃣ Crypto → 港股
| Crypto | 港股 |
|--------|------|
| BTC ↑ | OSL Group |
| ETH ↑ | Web3概念股 |

#### 2️⃣ 美股科技 → 港股科技
| 美股 | 港股 |
|------|------|
| NVDA ↑ | Tencent Holdings |
| QQQ ↑ | Alibaba Group |
| TSLA ↑ | Meituan |

#### 3️⃣ 风险情绪 → 高Beta股
| 情绪 | 港股 |
|------|------|
| 风险偏好 ↑ | 中小盘科技 |
| 风险偏好 ↓ | 防御股 |

