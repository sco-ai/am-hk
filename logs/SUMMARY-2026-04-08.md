#### Agent 2 - DataCurator (数据策展层)
**状态**: ✅ 完成 (v1.02)

**30+机构级因子计算**:
- 量价因子: 动量(5m/15m/1h)、波动率、流动性、换手率
- 技术指标: MA5/MA20、RSI14、MACD、布林带、ATR
- 盘口因子: 买卖压力、Order Imbalance、撤单率
- 资金流因子: 北水强度、主力净流入速度
- 跨市场因子: Crypto-美股-港股传导系数

**数据质量控制**:
- 异常检测: Z-score (>3σ标记)
- 缺失处理: 前向填充
- 延迟监控: ≤50ms
- 质量评分: 0-1

**跨市场信号融合**:
```
Final Score = 0.4×Crypto + 0.4×US + 0.2×HK_Local
```

**Kafka Topic**: `am-hk-processed-data`

---

#### Agent 3 - AlphaScanner (机会筛选层)
**状态**: ✅ 完成 (v1.03)

**多策略并行扫描**:
| 策略 | 权重 | 逻辑 |
|------|------|------|
| 动量策略 | 35% | 短期/中期动量 + 成交量确认 |
| 价值策略 | 20% | RSI超卖+布林带回归 |
| 情绪策略 | 25% | 主力资金+北水+散户比 |
| 跨市场 | 20% | Layer1先行+Layer2确认 |

**模型集成**:
- 主模型: LightGBM (云训练)
- 备选: XGBoost
- 策略优化: GPT-4.1 / DeepSeek

**机会分层**:
- Top 5: 核心重仓
- Top 6-10: 机会仓
- Top 11-20: 观察池

**Kafka Topic**: `am-hk-trading-opportunities`

#### Agent 5 - RiskGuardian (风控层)
**状态**: ✅ 完成 (v1.04)

**三层风控体系**:

| Layer | 类型 | 规则/模型 |
|-------|------|----------|
| 1 | 硬规则 | 单日≤2%、单笔≤0.5%、杠杆≤2x、持仓≤10只 |
| 2 | 动态规则 | 高波动→仓位50%、强趋势→100%、震荡→30% |
| 3 | 异常检测 | Isolation Forest (8维特征) |

**测试验证**: ✅ 全部通过

```
测试完整风控流程 - 通过
```

**Kafka Topic**: `am-hk-risk-approved-trades`

---

#### Agent 6 - LearningFeedback (进化层)
**状态**: ✅ 完成 (v1.04)

**核心模块**:

| 维度 | 模型 | 功能 |
|------|------|------|
| 策略优化 | LightGBM增量学习 | 因子权重动态调整 |
| 时序优化 | Informer微调 | 预测模型持续优化 |
| RL优化 | PPO + LSTM | 策略参数强化学习 |
| 图结构 | Temporal GNN | 市场关系图学习 |
| 仓位优化 | 凯利公式 | 最优仓位计算 |

**凯利公式**:
```
f* = (p×b - q) / b
其中: p=胜率, q=败率, b=盈亏比
```

**Kafka Topic**: `am-hk-model-updates` (广播给所有Agent)
