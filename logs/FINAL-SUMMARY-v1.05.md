## ✅ 7个Agent详细说明

### Agent 1: MarketHarvester (数据采集层)
**职责**: 多市场实时数据采集

| 市场 | 标的 | 数据类型 |
|------|------|----------|
| BTC | 5币种 | 资金费率、持仓量、多空比、K线、订单簿 |
| 港股 | 22只 | 北水资金(港股通)、Level2盘口、分时K线 |
| 美股 | 6只 | 盘前盘后、Dark Pool、ETF资金流、期权 |
| 新闻 | 多源 | 分类(财报/政策/行业/突发)、情绪评分 |

**输出**: `am-hk-raw-market-data`

---

### Agent 2: DataCurator (数据策展层)
**职责**: 数据清洗、因子计算、跨市场融合

**30+机构级因子**:
- 量价因子: 动量(5m/15m/1h)、波动率、流动性
- 技术指标: MA、RSI、MACD、布林带、ATR
- 盘口因子: 买卖压力、Order Imbalance
- 资金流因子: 北水强度、主力净流入

**数据质量控制**:
- Z-score异常检测 (>3σ标记)
- 延迟监控: ≤50ms
- 完整性: ≥99%

**输出**: `am-hk-processed-data`

---

### Agent 3: AlphaScanner (机会筛选层)
**职责**: 多策略扫描、因子评分、机会分层

**策略权重**:
| 策略 | 权重 | 逻辑 |
|------|------|------|
| 动量 | 35% | 短/中期动量 + 成交量确认 |
| 价值 | 20% | RSI超卖+布林带回归 |
| 情绪 | 25% | 主力资金+北水+散户比 |
| 跨市场 | 20% | Layer1先行+Layer2确认 |

**模型**: LightGBM (云训练) / GPT-4.1优化

**输出**: `am-hk-trading-opportunities` (Top5/Top10/Top20分层)

---

### Agent 4: TrendOracle (决策层)
**职责**: 双轨AI决策、止盈止损计算

**双轨架构**:
| Track | 权重 | 模型 |
|-------|------|------|
| A | 80% | Informer/Autoformer |
| B | 20% | Qwen2.5/Kimi |
| NLP | 并行 | FinBERT |

**输出**:
```json
{
  "action": "BUY",
  "confidence": 0.82,
  "entry": {"price": 65800},
  "tp": 66200, "sl": 64500,
  "position_size": 0.15
}
```

**输出**: `am-hk-trading-decisions`

---

### Agent 5: RiskGuardian (风控层)
**职责**: 三层风控、异常检测

| Layer | 类型 | 规则/模型 |
|-------|------|----------|
| 1 | 硬规则 | 单日≤2%、单笔≤0.5%、杠杆≤2x |
| 2 | 动态规则 | 高波动→50%仓位、强趋势→100% |
| 3 | 异常检测 | Isolation Forest |

**测试**: ✅ 全部通过

**输出**: `am-hk-risk-approved-trades`

---

### Agent 6: LearningFeedback (进化层)
**职责**: 策略持续优化、模型更新

| 模块 | 模型 | 功能 |
|------|------|------|
| 策略优化 | LightGBM增量 | 因子权重调整 |
| 时序优化 | Informer微调 | 预测优化 |
| RL优化 | PPO+LSTM | 策略参数学习 |
| 图结构 | Temporal GNN | 市场关系学习 |
| 仓位优化 | 凯利公式 | f* = (p×b-q)/b |

**输出**: `am-hk-model-updates` (广播给所有Agent)

---

### Agent 7: PerformanceAnalyzer (绩效分析层) ⭐ 本次新增
**职责**: 绩效评估、策略归因、报告生成

**绩效指标**:
- 收益率、夏普比率、索提诺比率
- 最大回撤、卡尔马比率
- 胜率、盈亏比、波动率
- VaR(95%)、CVaR、Alpha、Beta

**策略归因**:
- 各策略贡献度分析
- 胜率/交易数统计
- 自动优化建议生成

**输出**: `am-hk-performance-reports`
